1nf First Normal Form

If a relation contains any type of multi value or composite attributes it is violation of first normal form (1NF). The relation is in first normal form if it does contain any multi value or composite attributes. In simple terms a relation is in first normal form if every attribute in that relation is singled valued atomic attribute.

A relation in 1NF

  • There is only single valued attributes
  • Attribute domain does not change
  • There is a unique name for every attribute/column
